# sprint_7_software_development_tools
Final project of sprint 7 (tripleten academy)
Este proyecto es un análisis exploratorio de datos de vehículos en Estados Unidos.
Este proyecto tiene como objetivo explorar los ambientes de exploración virtuales en Python, construrir el mío propio y lanzar mi app con ayuda de Render, todo esto para desarrollar habilidades escenciales en el análisis de datos avanzado.
Este proyecto es simple y sencillo, pero es el primero de muchos en mi carrera como Científico de Datos.
